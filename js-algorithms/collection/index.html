Set:

The Set object lets you store unique values of any type, whether primitive values or object references. add(value): Add a new element to Set;

Methods
remove(value): Remove an element from Set;
has(value): Return true if the value in the Set, overwise false;
clear(): Remove all elements from the Set;
size(): Return the amount of elements in the Set;
values(): Return an Array concluding all the elements;
Collections Methods
union(otherSet): 返回该集合与参数集合的并集
intersection(otherSet)：返回该集合与参数集合的交集
difference(otherSet)：返回该集合与参数集合的差集（该集合有而参数集合没有）
sebset(otherSet)：返回布尔值--该集合是否是参数集合的子集
